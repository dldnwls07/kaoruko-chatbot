# 🚀 개발자 치트시트 (빠른 참조용)

## ⚡ 즉시 사용 가능한 명령어

```bash
# 호감도 100 만들기
/dev maxaffection
/dev 호감도100

# 호감도 특정 값으로 설정
/dev affection 50    # 호감도 50
/dev affection -20   # 호감도 -20
/dev 호감도 80        # 호감도 80

# 원하는 코인 받기 (1~100,000)
/dev coins500      # 500코인
/dev coins1000     # 1000코인  
/dev coins10000    # 10,000코인
/dev coins99999    # 99,999코인

# 모든 테마 아이템 받기
/dev allitems  
/dev 모든아이템

# 도움말 보기
/dev help
/dev 도움말
```

## 📋 자주 사용하는 조합

### 🎯 완전 개발 모드 설정
1. `/dev maxaffection` (호감도 최대)
2. `/dev coins1000` (코인 충전)
3. `/dev allitems` (모든 아이템)

### 🧪 테스트 환경 구축
- 호감도 100으로 설정하여 모든 대화 옵션 테스트
- 충분한 코인으로 상점 시스템 테스트![1762317195367](image/CHEAT_SHEET/1762317195367.png)![1762317250389](image/CHEAT_SHEET/1762317250389.png)

## 💡 사용 팁

- **즉시 적용**: 명령어 입력 후 엔터만 누르면 바로 적용
- **캐릭터 반응**: 호감도 변경 시 캐릭터가 상황을 인지하고 반응
- **감정 동기화**: 호감도에 맞는 감정 상태로 자동 변경
- **애니메이션**: 변화가 시각적으로 표시됨
- **확인**: 초록색 시스템 메시지로 완료 확인
- **조합**: 여러 명령어를 연속으로 사용 가능

## 🎨 시각 효과

- ✅ **성공**: 초록색 글로우 메시지
- 📈 **호감도**: 하트 애니메이션 + 숫자 증가  
- 🪙 **코인**: 코인 아이콘 반짝임 + 카운트 업
- 🎭 **테마**: 인벤토리에서 즉시 확인 가능
