 🛠️ 개발자 명령어 가이드

## 📋 개요
이 문서는 채팅봇 개발 및 테스트를 위한 개발자 전용 치트 명령어들을 정리한 것입니다.

## ⚡ 사용 방법
채팅창에서 다음 명령어들을 입력하면 즉시 적용됩니다.

---

## 🎯 호감도 관리

### 호감도 최대치 설정
- `/dev maxaffection` - 호감도를 100으로 즉시 설정
- `/dev max호감도` - 한글 버전
- `/dev 호감도100` - 간단 버전

### 호감도 특정 값 설정
- `/dev affection [숫자]` - 호감도를 특정 값으로 설정 (-100~100)
- `/dev 호감도 [숫자]` - 한글 버전

**효과:**
- 호감도가 즉시 지정한 값으로 변경됨
- 진행바와 하트가 올바르게 동기화됨
ㅌ- **캐릭터의 감정 상태 자동 업데이트** (사랑, 친밀함, 호감, 관심, 평범, 불편함)
- **캐릭터가 호감도 변화를 인지하고 반응** (상황에 맞는 대사 출력)
- 호감도 변화 애니메이션 표시
- localStorage에 즉시 저장
- 시스템 메시지로 완료 알림

---

## 💰 코인 관리

### 코인 지급
- `/dev coins1000` - 1000코인 즉시 지급
- `/dev 코인1000` - 한글 버전

**효과:**
- 현재 코인에 1000코인 추가
- 총 누적 코인도 함께 증가
- 코인 획득 애니메이션 표시
- 시스템 메시지로 완료 알림

---

## 🎨 아이템 관리

### 모든 아이템 획득
- `/dev allitems` - 모든 테마 아이템 즉시 획득
- `/dev 모든아이템` - 한글 버전

**효과:**
- 모든 테마 아이템 (보라색, 파란색, 초록색, 주황색) 획득
- 인벤토리에서 즉시 확인 가능
- 시스템 메시지로 완료 알림

## 🎨 시각적 특징

### 시스템 메시지 스타일
개발자 명령어를 통한 시스템 메시지는 다음과 같은 특별한 스타일로 표시됩니다:

- **색상**: 초록색 그라디언트 (해커 스타일)
- **폰트**: Courier New (모노스페이스)
- **효과**: 글로우 애니메이션
- **위치**: 중앙 정렬

### 애니메이션 효과
- **호감도 변화**: 숫자 증가 애니메이션 + 하트 효과
- **코인 획득**: 코인 카운트 증가 애니메이션
- **시스템 메시지**: 글로우 효과로 깜빡임

## 🔧 기술적 세부사항

### 명령어 처리 로직
```javascript
// 개발자 명령어는 handleSendMessage 함수에서 처리됨
// '/dev'로 시작하는 메시지를 감지하여 특별 처리
// 일반 채팅 API 호출 없이 즉시 로컬에서 처리
```

### 상태 변경
- **호감도**: `setAffectionLevel(100)`
- **코인**: `setCoins(prev => prev + 1000)`
- **아이템**: `setOwnedItems(['default', 'purple', 'blue', 'green', 'orange'])`

## ⚠️ 주의사항

1. **개발 환경 전용**: 이 명령어들은 개발 및 테스트 목적으로만 사용
2. **데이터 초기화**: 브라우저 새로고침 시 localStorage에 저장된 데이터로 복원
3. **서버 동기화**: 개발자 명령어는 서버와 동기화되지 않음 (로컬 상태만 변경)
4. **디버깅 도구**: 기능 테스트 및 디버깅을 위한 도구로 활용
