# 🌸 카오루코 챗봇 프로젝트

와구리 카오루코와 대화할 수 있는 챗봇입니다.
「향기로운 꽃 늠름하게 핀다」의 주인공 카오루코와 자연스러운 대화를 나누세요!

## 🚀 실행 방법

### ⚡ 간단 실행 (권장)

프로젝트 폴더에서 **`챗봇_실행.bat`** 파일을 더블클릭하거나 터미널에서 실행하세요:

```bash
챗봇_실행.bat
```

자동으로 다음이 실행됩니다:
1. **백엔드 서버 시작** (포트 8001) - Python FastAPI
2. **프론트엔드 서버 시작** (포트 5173) - React + Vite  
3. **웹브라우저 자동 실행** - http://localhost:5173

### 🛑 서버 종료
**`챗봇_종료.bat`** 파일로 모든 서버를 안전하게 종료할 수 있습니다.

### 💬 사용법
1. 처음 실행하면 이름을 입력하세요
2. 카오루코와 자유롭게 대화하세요
3. 카오루코는 과거 대화를 기억합니다

### 🎭 고급 감정 시스템
- **6가지 핵심 감정**: 수줍음, 기쁨, 슬픔, 화남, 놀람, 설렘
- **실시간 감정 분석**: AI가 대화 내용을 분석하여 자연스러운 감정 표현
- **감정별 UI 변화**: 감정에 따른 색상과 이모지 변화
- **감정 강도 시스템**: 1-10 단계의 세밀한 감정 표현

### 💕 다층적 호감도 시스템  
- **7단계 관계 발전**: 멀어진사람 ← 낯선사람 → 지인 → 친구 → 친한친구 → 절친 → 연인
- **실시간 호감도 변화**: -100 ~ +100 범위에서 대화에 따른 즉시 반영
- **관계별 말투 변화**: 친밀도에 따른 자연스러운 대화 톤 변화
- **시각적 호감도 표시**: 
  - 💖 양수: 분홍 하트와 왼쪽→오른쪽 게이지
  - 💔 음수: 깨진 하트와 오른쪽→왼쪽 빨간 게이지

### 🎮 미연시 게임 요소
- **호감도 이정표**: 특정 점수 도달 시 특별 이벤트 및 보상
- **기념일 시스템**: 첫 만남 기준 날짜 계산 및 특별 메시지
- **관계 발전 애니메이션**: 단계 상승 시 축하 효과

### 🛒 상점 & 경제 시스템
- **코인 시스템**: 대화를 통한 코인 획득 (호감도별 차등 지급)
- **다양한 아이템**: 호감도 부스터, 테마, 직접 증가 아이템
- **테마 상점**: 4가지 색상 테마 (보라, 파랑, 초록, 주황) 구매 가능
- **부스터 아이템**: 일정 시간 동안 호감도 획득량 증가
- **구매 확인 시스템**: 상세한 가격 정보와 잔액 표시

### 🎒 인벤토리 관리
- **아이템 보관**: 구매한 모든 아이템 저장 및 관리
- **테마 적용**: 보유한 테마를 실시간으로 적용/변경 가능
- **사용 내역**: 아이템 사용 기록 및 상태 표시

### 💭 자연스러운 대화 시스템
- **속마음 표현**: `*내용*` 형태의 솔직한 감정 드러내기
- **컨텍스트 기억**: 과거 대화 내용을 기억하고 연관성 유지
- **캐릭터별 말투**: 각 캐릭터만의 독특한 어조와 반응 패턴

## 🛠 기술 스택

### � Backend
- **Python 3.11+** - 메인 개발 언어
- **FastAPI** - 고성능 웹 프레임워크
- **SQLite** - 경량 데이터베이스 (사용자 데이터, 호감도 등)
- **Google Gemini API** - AI 대화 생성

### 🎨 Frontend  
- **React 18** - UI 라이브러리
- **Vite** - 빠른 개발 서버 및 빌드 도구
- **CSS3** - 커스텀 스타일링 (캐릭터별 테마)
- **JavaScript ES6+** - 모던 자바스크립트

### 📋 설치 요구사항
- **Python 3.11** 이상
- **Node.js 16** 이상
- **Google Gemini API 키** (대화 AI 처리용)

### 🔑 API 키 설정
1. [Google AI Studio](https://makersuite.google.com/app/apikey)에서 무료 API 키 발급
2. `backend/.env.template` → `backend/.env`로 복사
3. `.env` 파일에서 `YOUR_API_KEY_HERE`를 발급받은 API 키로 교체

```env
GEMINI_API_KEY=your_actual_api_key_here
```

### 🛠 문제 해결
- **API 키 오류**: `backend/.env` 파일의 API 키 확인
- **포트 충돌**: 8001(백엔드), 5173(프론트엔드) 포트 확인
- **패키지 오류**: 
  ```bash
  pip install -r backend/requirements.txt
  cd frontend && npm install
  ```
