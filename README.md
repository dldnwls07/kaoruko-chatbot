# 🌸 카오루코 챗봇 프로젝트

와구리 카오루코와 대화할 수 있는 챗봇입니다.
「향기로운 꽃 늠름하게 핀다」의 주인공 카오루코와 자연스러운 대화를 나누세요!

## 🚀 실행 방법

### ⚡ 간단 실행 (권장)

프로젝트 폴더에서 **`카오루코_실행.bat`** 파일을 더블클릭하거나 터미널에서 실행하세요:

```bash
카오루코_실행.bat
```

그러면 자동으로:
1. 백엔드 서버 시작 (포트 8001)
2. 프론트엔드 서버 시작 (포트 5173)
3. 웹브라우저가 자동으로 열림

### 🌐 접속
웹브라우저에서 `http://localhost:5173` 접속

### 💬 사용법
1. 처음 실행하면 이름을 입력하세요
2. 카오루코와 자유롭게 대화하세요
3. 카오루코는 과거 대화를 기억합니다

### 🎭 고급 감정 시스템
- **6가지 핵심 감정**: 수줍음, 기쁨, 슬픔, 화남, 놀람, 설렘
- **실시간 감정 분석**: 대화 내용을 분석하여 자연스러운 감정 표현
- **감정별 UI 변화**: 감정에 따른 색상과 이모지 표시
- **감정 강도**: 1-10 단계로 세밀한 감정 표현

### 💕 호감도 시스템  
- **7단계 관계 발전**: 멀어진사람 ← 낯선사람 → 지인 → 친구 → 친한친구 → 절친 → 연인
- **양방향 호감도 변화**: -100 ~ +100 범위의 실시간 호감도 증감
- **관계별 말투**: 친밀도에 따른 대화 톤과 호칭 변화
- **시각적 피드백**: 
  - 양수: 💖 분홍 하트와 왼쪽→오른쪽 게이지
  - 음수: 💔 깨진 하트와 오른쪽→왼쪽 빨간 게이지

### 🎮 미연시 이벤트 시스템
- **호감도 이정표**: 특정 점수 달성 시 특별 이벤트 발생
- **특별 대화**: 관계 단계별 고유 대화와 스토리
- **기념일 시스템**: 첫 만남 기준 날짜 계산 및 기념
- **이벤트 알림**: 관계 발전 시 축하 메시지와 애니메이션

### 💭 속마음 표현 시스템
- **자연스러운 속마음**: `*내용*` 형태의 솔직한 감정 표현
- **시각적 구분**: 괄호 표시와 투명도로 속마음과 일반 대화 구분
- **상황별 활용**: 수줍거나 부끄러울 때의 진솔한 감정 드러내기

### 📋 사전 준비사항
- **Python 3.11** 이상
- **Node.js** 
- **Google Gemini API 키** (대화 처리용)

### 🔑 API 키 설정 방법
1. [Google AI Studio](https://makersuite.google.com/app/apikey)에서 API 키 발급
2. `backend/.env.template` 파일을 `backend/.env`로 복사
3. `.env` 파일에서 `YOUR_API_KEY_HERE`를 실제 API 키로 교체

### 🛠 문제 해결
- **API 키 오류**: `backend/.env` 파일 확인
- **포트 충돌**: 8001, 5173 포트가 사용 중인지 확인
- **패키지 오류**: `pip install -r requirements.txt`, `npm install` 재실행
