# 🌸 카오루코 챗봇 프로젝트

와구리 카오루코와 대화할 수 있는 AI 챗봇입니다.

## 🚀 실행 방법

### ⚡ 간단 실행 (권장)

프로젝트 폴더에서 **`카오루코_실행.bat`** 파일을 더블클릭하거나 터미널에서 실행하세요:

```bash
카오루코_실행.bat
```

그러면 자동으로:
1. 백엔드 서버 시작 (포트 8001)
2. 프론트엔드 서버 시작 (포트 5173)
3. 웹브라우저가 자동으로 열림

### 🌐 접속
웹브라우저에서 `http://localhost:5173` 접속

### 💬 사용법
1. 처음 실행하면 이름을 입력하세요
2. 카오루코와 자유롭게 대화하세요
3. 카오루코는 과거 대화를 기억합니다

### 🎤 AI 음성 클로닝 기능
- **실제 와구리 카오루코 목소리**로 대화 가능
- **감정별 음성 변화**: 수줍음, 기쁨, 슬픔, 화남, 놀람, 설렘
- **자동 음성 재생**: 메시지마다 자동으로 음성 출력
- **음성 파일 준비**: `backend/ai_tts/audio_data/와구리.mp3` 파일 배치 필요

### 📋 사전 준비사항
- **Python 3.11** 이상
- **Node.js** 
- **Google AI API 키** (발급 및 설정 필요)

### 🔑 API 키 설정 방법
1. [Google AI Studio](https://makersuite.google.com/app/apikey)에서 API 키 발급
2. `backend/.env.template` 파일을 `backend/.env`로 복사
3. `.env` 파일에서 `YOUR_API_KEY_HERE`를 실제 API 키로 교체

### 🛠 문제 해결
- **API 키 오류**: `backend/.env` 파일 확인
- **포트 충돌**: 8001, 5173 포트가 사용 중인지 확인
- **패키지 오류**: `pip install -r requirements.txt`, `npm install` 재실행
